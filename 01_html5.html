지각생의 웹세상. HTML5 월드

  1. HTML5의 신기능

    1. 오프라인 웹 애플리케이션 - 네트워크가 연결되지 않은 상태에서도 실행 가능
    2. 드래그 앤 드롭 - 요소들을 마우스로 끌어서 넣을 수 있음
    3. 웹 스토리지 - 쿠키를 대체할 수 있는 웹 저장소 기능 제공
    4. 위치 정보 제공 - 지도 기능
    5. 웹 SQL 데이터베이스
    6. 파일 API 지원 - 파일 업로드와 파일 관리 기능 제공
    7. 웹소켓 API제공 - 서버와 브라우저 간의 양방향 통신 기능 제공

  2. 기본 태그들


    1. Atom에서 기본적으로 작성해주는 태그들(2021년 기준)
    2. !DOCTYPE html : 관용적 표현. 이 문서는 html에 있다란 의미.
    3. html : head태그와 body태그를 포괄하는 태그.
    4. head : body를 설명하는 태그들(meta,title)은 head태그로 묶기로 약속함.
    5. meta : utf-8로 작성된 아톰 문서를 웹브라우저에서도 utf-8로 읽도록 지정하는 기능.meta charset="utf-8"
    6. title : 제목. 작성시 웹페이지 최상단 꼬리표에 명시되는 타이틀.
    7. body : 본문 내용은 body태그 사이에 넣기로 약속함.

      • type="square"

    11. ol : ordered list. ul기능에서 자동번호매김기능 추가.
      • type="1" <<--기본값
      • type="I" <<--로마 대문자
      • type="i" <<--로마 소문자
      • type="A" <<--알파벳 대문자
      • type="a" <<--알파벳 소문자
      • 예시 : <ol type="1" start="100"> </ol>
      • 예시 설명 : 100번 번호부터 시작해서 "1"타입으로 번호를 매겨라.
